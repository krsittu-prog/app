## 📞 QUICK REFERENCE

### If App Won't Load
- Press `r` to reload
- Close and reopen
- Restart server: `npm start`

### If Video Won't Play
- Check internet
- Verify video is public
- Try uploading local video

### If PDF Not Showing
- Refresh page
- Check backend running
- Try new PDF upload
